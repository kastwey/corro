The generic earcons (`draw.ogg` = `card.draw`, `discard.ogg` = `card.discard`, `shuffle.ogg` =
`cards.shuffle`, `turn-you.ogg`, `game-over.ogg`) come from
[`sound-commons/`](../../../sound-commons/ATTRIBUTIONS.md). Draw and reshuffle reuse those shared
cues. Every action play uses the rising family warning while it can still be Noped.
